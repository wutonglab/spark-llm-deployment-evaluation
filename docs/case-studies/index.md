## Contribute

Use the [case-study template](https://github.com/wutonglab/spark-llm-deployment-evaluation/blob/main/case-studies/_template.md). Cases must:

- Be sanitized (no internal hostnames, IPs, customer business data)
- Cite ≥ 1 channel source per framework dimension
- Include reproducible launcher + measurements
